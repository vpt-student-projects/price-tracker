PriceTracker - Система для мониторинга цен
Курсовой проект по дисциплине МДК 02.02 "Инструментальные средства разработки программного обеспечения"

О проекте
Система автоматического отслеживания цен на товары в интернет-магазинах с ведением истории и аналитикой.

Разработчики: Карабеков Дияр, Поздняков Сергей

Функциональность
- Добавление товаров по ссылке из магазинов

- Автопарсинг цен каждые 5 минут

- История изменений с графиками

- Статистика (мин/макс/средняя цена)

- Desktop приложение на WPF

Технологии
Backend: ASP.NET Core, Entity Framework, SQLite

Frontend: WPF, OxyPlot (графики)

Парсинг: HttpClient, AngleSharp

Поддерживаемые магазины
DNS-Shop

Citilink

OnlineTrade

Books.toscrape.com 

API Endpoints
GET /api/products - список товаров

POST /api/products - добавить товар

GET /api/stats/overview - статистика

POST /api/products/{id}/refresh - обновить цену
